# Stock Trading Platform (Java)

A simple console-based Stock Trading Platform built using Java.  
This project allows users to buy and sell stocks, view portfolio details, and manage balance.

## Features

- View Market Data
- Buy Stocks
- Sell Stocks
- View Portfolio
- Check Account Balance
- Simple Console Interface

## Technologies Used

- Java
- HashMap
- OOP Concepts
- Scanner Class

## Available Stocks

| Stock Name | Price |
|------------|--------|
| TCS | ₹3500 |
| INFY | ₹1500 |
| RELIANCE | ₹2500 |

## How It Works

1. User starts with an initial balance of ₹10000.
2. User can:
   - Buy stocks
   - Sell owned stocks
   - View portfolio
   - Check remaining balance
3. Portfolio updates automatically after transactions.

## Project Structure

- `Stock` class → Stores stock name and price
- `StockTradingPlatform` class → Main logic of the application
